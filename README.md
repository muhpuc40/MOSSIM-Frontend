# MOSSIM

🌐 Website: https://mossim.net

MOSSIM is a modern e-commerce web application built with **Next.js**.  
It is designed for showcasing and selling fashion products with a clean, responsive, and user-friendly interface.

---

## 🚀 Features

- Modern UI/UX design
- Responsive layout (mobile & desktop)
- Product listing & detail pages
- Shopping cart functionality
- Wishlist system
- Checkout pages
- Multiple product layouts and styles
- Built with reusable components

---

## 🛠️ Tech Stack

- Next.js 14
- React
- TypeScript
- SCSS
- Tailwind CSS (optional usage)
- Context API (state management)

---

## ⚙️ Installation & Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
